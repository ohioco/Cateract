# Cateract

Cateract is a custom private web network system built with Node.js and Express.  
It simulates a simplified internet environment with custom routing and a browser-style UI.

## Features
- Custom site system (`/site/:name`)
- Internal network pages
- Basic browser UI with `cat://` navigation concept
- Local network hosting support

## Setup

```bash
git clone https://github.com/ohioco/Cateract.git
cd Cateract
npm install
npm start
